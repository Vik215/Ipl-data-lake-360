# IPL Data Lake 360

## Project Overview

IPL Data Lake 360 is an end-to-end AWS Data Engineering project built using a Medallion Architecture (Bronze, Silver, Gold).

The project demonstrates how to build a serverless data pipeline that ingests IPL cricket data, validates it, transforms it, and generates business-ready analytics.

## Tech Stack

- AWS S3
- AWS Lambda
- AWS Step Functions
- AWS Glue
- AWS Glue Data Catalog
- Amazon Athena
- Amazon CloudWatch
- IAM
- Python
- PySpark
- SQL
- Terraform (coming soon)
